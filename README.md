### Overview :
Hostel Hub is a comprehensive Android application designed to streamline hostel management for both students and wardens. It offers features like Google login, a NoticeBoard, QR-based attendance, and more, all built with MVVM architecture and integrated with Room and Firebase databases.
### Features :
* Google Login and Registration for both Students and Wardens 
* Dark mode
* NoticeBoard for Announcements and Updates
* QR-Based Attendance System for Quick and Accurate Tracking
* Separate Fragments for Students and Wardens
* Secure Data Management using Room and Firebase databases

### Getting Started :
Note: To run the app, you need to configure Firebase and generate Google Services JSON.
 * Set up Firebase for your project and download the google-services.json file.
 *  Place the google-services.json file in the app/ directory.
 * Rebuild app
